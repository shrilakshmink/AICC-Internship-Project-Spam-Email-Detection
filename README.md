Email Spam Detection Web App using Machine Learning and Streamlit

## 📧 Email Spam Detection Web App
## 🌐 Live Demo

Built an interactive spam detection system using Machine Learning.

### 🔹 Features
- Detects spam from full email text
- Uses TF-IDF vectorization with n-grams
- Machine Learning model (Naive Bayes / Logistic Regression)
- Interactive web app using Streamlit

### 🔹 Example
Input: "Congratulations! You won a free iPhone"  
Output: Spam

### 🔹 Accuracy
Achieved ~85–95% accuracy using improved dataset and preprocessing.

### 🔹 Tech Stack
- Python
- Scikit-learn
- Streamlit
